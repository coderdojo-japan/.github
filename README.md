# 一般社団法人 CoderDojo Japan のメタ設定一覧

本リポジトリでは[一般社団法人 CoderDojo Japan](https://coderdojo.jp/about-coderdojo-japan) の GitHub アカウント ([@coderdojo-japan](https://github.com/coderdojo-japan)) 内にある[各種リポジトリ](https://github.com/coderdojo-japan)のデフォルト値などを設定されています。個別リポジトリ内に同名のファイルを置けば上書きも可能です。

- GUIDELINE: https://github.com/coderdojo-japan/.github/blob/main/CODE_OF_CONDUCT.md
- SECURITY: https://github.com/coderdojo-japan/.github/blob/main/SECURITY.md
- FUNDING: https://github.com/coderdojo-japan/.github/blob/main/FUNDING.yml

English: This repository defines default settings above for [GitHub repositories](https://github.com/coderdojo-japan) in CoderDojo Japan Association ([@coderdojo-japan](https://github.com/coderdojo-japan)). These default settings can be overriden by putting corresponding files in each repopository.
